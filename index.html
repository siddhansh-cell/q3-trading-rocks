Functional rocks
What is a rock?

A rock is a step-change you build once this quarter that permanently moves a dial. Continuous work — margin, new customers, trading the UK engine — lives on the scorecard. Rocks are the change you drive on top. Done or not done. Binary.

01 Measurement layer Data team
Description
Build the unified measurement layer across NCs, ECs, and margin. Fully automated with live updates — actions flow from data to decisions without manual pulls.
Metric
NC attribution accuracy; margin decomposition (mix, cost, price, discounts); EC retention metrics live
Definition of done
All three layers (NC, EC, margin) live in Snowflake with automated daily refresh. No manual data pulls needed to run the weekly trade meeting.
02 NC spend funnel Leo
Description
Performance marketing spend funnelled efficiently into new customers. Scale through LTV bidding rollout, NC-only bidding, customer segmentation, and increased test velocity with Mai.
Metric
New customers (41k→50k ramp); CAC; NCPP; PM as % of turnover
Definition of done
LTV bidding rolled out across campaigns. NC-only bidding tested and scaled. Mai test velocity at target cadence. NC monthly ramp on track.
03 Search + site discoverability Arman
Description
Search as the engine of conversion on product. Improve site navigation, filtration, and product discovery to drive CRO.
Metric
Search conversion rate; filtration usage; product discovery metrics (TBC)
Definition of done
Search reranking live (fidelity vs commercial balance defined). Navigation filtration shipped. Measurable CRO uplift on search-driven sessions.

Description
Launch store credit to drive 1→2 reactivation. Set the foundations for a broader loyalty programme that increases repeat rate and LTV.
