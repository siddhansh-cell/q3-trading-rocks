Functional rocks
What is a rock?

A rock is a step-change you build once this quarter that permanently moves a dial. Continuous work - margin, new customers, trading the UK engine - lives on the scorecard. Rocks are the change you drive on top. Done or not done. Binary.

01 Measurement layer Siddhansh + Jay
Description
Build unified measurement layers across NC, EC, brand movements, and margin - metrics that capture if the business is truly improving. Fully automated with live updates - actions flowing from data to decisions without manual pulls. Diagnostic layer for growth trading.
Definition of done
All four layers (NC, EC, brand, margin) live in Snowflake with automated daily refresh.
02 NC spend funnel Siddhansh + Leo
Description
Funnel performance marketing spend into acquiring new customers. Multiple approaches available:
  • LTV bidding
  • NC-only bidding
  • Customer segmentation
  • Testing velocity with Mai
Metric
New customers: 35k current, ramp to 45k by September. NCPP is the headline metric.
Definition of done
  • LTV bidding rolled out across campaigns
  • NC-only bidding tested and scaled if successful
  • Monthly ramp on track
03 Search + site discoverability Siddhansh + Arman
Description
  • Search as an engine of conversion
  • Search results are accurate
  • Improve site navigation and filtration
Metric
Improvement in search conversion rate. Usage of filtration on website.
Definition of done
  • Search reranking live
  • Fidelity, discovery, and commercial balance maintained
  • Navigation filter shipped (e.g. vegan)
  • Measurable CRO uplift on search-driven sessions
04 Store credit + loyalty foundations Siddhansh + Holly
Description
Launch store credit to drive 1-to-2 reactivation. Set the foundations for a broader loyalty programme that increases repeat rate and LTV.
Metric
1-to-2 repeat rate. Store credit redemption rate.
Definition of done
Store credit mechanism live and tested with 3 use cases. Loyalty programme roadmap defined for Q4.
05 Pricing automation Siddhansh + Anton
Description
Move from manual pricing to automated engine. Price scraper integrated into Snowflake with clear metrics for lowest/cheapest price.
Metric
Pricing gain %. EN-UK/.com price convergence within 2-2.5%.
Definition of done
Scraper covers full scope with 90%+ adoption. Target: end of July.
06 CW inbound calls Siddhansh + Harriet
Description
Pilot inbound call handling in Customer Wellbeing. Understand if it helps deliver better customer experience.
Metric
Inbound call CSAT 4.8+. 10% adoption.
Definition of done
Inbound calls piloted. Scaled to ~10% of total tickets with focus on VIPs.
